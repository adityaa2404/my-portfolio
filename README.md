# Aditya Potdar — Next.js Portfolio (Twitter/X Inspired)

A futuristic, timeline-style developer portfolio built with **Next.js App Router**, React, Framer Motion, and a Twitter/X-inspired layout.

## Run locally

```bash
npm install
npm run dev
```

Open: `http://localhost:3000`

## Build

```bash
npm run build
npm run start
```

## Routes supported

- `/`
- `/projects`
- `/notifications`
- `/bookmarks`
- `/skills`
- `/profile`

All routes render the portfolio shell with route-specific content.
